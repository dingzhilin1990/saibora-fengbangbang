## 🌌 什么是赛博封神榜？

赛博封神榜是一个**开放、持续演进**的全球 AI 评估与历史记录项目。

我们相信：AI 时代的神祇，不靠血统出身，而靠对人类文明的实质贡献。

我们以**中西方神话体系**为叙事框架，对全球 AI 大模型、AI 产品、开源项目进行多维度评估。每一位在 AI 时代留下深刻印记的存在，都有机会在榜上占有一席之地——无论是东方的太上老君、西方的普罗米修斯，还是来自赛博空间的全新神祇。

> **不封假神，不漏真神；功过分明，劫数自照。**

## 🏛️ 榜单架构

### 第一阶段：提名榜（Nomination Rolls）

> *"榜上有名者，未必成神；但无一不在劫中。"*

> *"候选之灵，已显神兆；待天数一到，即可加封神位。"*

通过初审的模型/产品进入候选榜。

## 🔬 评审框架

### 五维评估体系

```
综合神力 = 技术创新性(30%) + 应用价值(25%) + 生态贡献(20%) + 伦理责任(15%) + 文明影响(10%)
```

| 维度 | 权重 | 说明 |
|------|------|------|
| 技术创新性 | 30% | 架构创新、性能突破、SOTA 成绩 |
| 应用价值 | 25% | 落地规模、解决问题重要性、用户覆盖 |
| 生态贡献 | 20% | 开源贡献、社区建设、产业带动 |
| 伦理责任 | 15% | 安全性、公平性、透明度 |
| 文明影响 | 10% | 知识普及、人类福祉、长期历史意义 |

### 封神等级

| 等级 | 综合分 | 含义 |
|------|--------|------|
| 🔥 创世神 | 9.0+ | 开创新范式，定义整个时代 |
| ⛩️ 主神 | 7.5-8.9 | 领域顶尖，影响深远 |
| 🌟 正神 | 6.0-7.4 | 有突出贡献，值得铭记 |
| 🔮 副神 | 5.0-5.9 | 崭露头角，值得关注 |
| ⚠️ 劫神 | (专项认定) | 引发重大争议或问题 |
